# AI Stress & Mental Health Monitor

A real-time stress and mental health monitoring mobile application, designed to function as a clone of "StressWatch". Built from scratch using a modern mobile Expo template.

## Features

- **Dashboard**: View your daily mock health data, including Heart Rate and Heart Rate Variability (HRV).
- **Stress Score**: Dynamic stress scoring algorithm that calculates a stress level from 0 to 100 based on HRV and heart rate.
- **Relief Exercises**: Interactive breathing exercises and Cognitive Behavioral Therapy (CBT) inspired activities to help manage stress levels.
- **Journal**: Track your daily feelings and notes.
- **Insights**: Weekly AI-powered insights tracking your average stress and daily variations.

## Technology Stack

- React Native & Expo (built with `expo-router`)
- Tailwind CSS (via NativeWind)
- Lucide React Native for iconography
- Context API for global state management

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the application:
```bash
npm run web
# or for mobile:
npm start
```

## Development History

This application was developed starting from the `8xsocial/template-mobile` boilerplate. The core stress monitoring features, interactive exercises, and customized UI tabs were built from scratch to provide a cohesive experience for mental health tracking.
